✅ АВТОМАТИЧЕСКИЕ (система подставляет сама):
Эти переменные НЕ НУЖНО вводить при тесте, система добавляет их автоматически:
{season}          # Текущий сезон → "зима", "весна", "лето", "осень"{year}            # Текущий год → 2025{current_year}    # То же что year → 2025{category}        # Категория блога → "Психология", "Красота" и т.д.{weekday}         # День недели (если указана дата) → "понедельник", "вторник" и т.д.{title}           # Формируется как "{zodiac_sign} прогноз" (если есть знак зодиака)
Код формирования автоматических переменных (строка 1724 в Asistent/views.py):
auto_vars = {    'season': season,              # Определяется по текущему месяцу    'year': now.year,              # Текущий год    'current_year': now.year,      # Дубль для удобства    'category': template.blog_category.title if template.blog_category else 'Блог',    'title': test_variables.get('zodiac_sign', 'Статья') + ' прогноз',    'weekday': weekday_ru,         # Парсится из даты (если указана)}
